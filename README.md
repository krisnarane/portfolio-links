# Portfolio 2026

Um portfólio de links pessoal moderno e profissional com design minimalista em tons de preto e magenta.

## 🎨 Características

- **Design Responsivo**: Funciona perfeitamente em desktop, tablet e mobile
- **Bilíngue**: Suporte para Português e Inglês com alternância dinâmica
- **Animações Suaves**: Gradientes fluidos e animações CSS modernas
- **Tela de Carregamento**: Loading screen profissional com barra de progresso
- **Social Links**: Links para LinkedIn, GitHub, X e Instagram
- **Foto de Perfil**: Circle animado com efeito flutuante
- **Sobre Mim**: Seção de informações pessoais

## 🚀 Como Usar

### Opção 1: Python HTTP Server (Recomendado)
```bash
python -m http.server 8000
```
Acesse: `http://localhost:8000`

### Opção 2: Abrir no Navegador
Simplesmente abra o arquivo `index.html` no seu navegador.

### Opção 3: Node.js
```bash
npx http-server
```

## 🛠️ Tecnologias

- **HTML5**: Estrutura semântica
- **CSS3**: Animações, gradientes, design responsivo
- **JavaScript (Vanilla)**: Sistema de i18n, loading screen, preferências
